## Welcome to the Tesco Mobile Design system

In here we will define a set of reusable styles that are platform agnostic, allowing for portability and ease of use. This should allow developers to focus on on the features and not have to worry about the styling.

We hope to extend on this design library to include as much of the ecosystem as possible.

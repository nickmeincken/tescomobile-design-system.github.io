## Padding

- `m` - for classes that set `margin`
- `p` - for classes that set `padding`

direction
- `t` = top
- `b` = bottom
- `l` = left
- `r` = right
- `x` = left & right
- `y` = top & bottom

sizing

- `0` = 0
- `1` = 0.5rem
- `2` = 1rem
- `3` = 2rem
- `4` = 4rem
- `5` = 8rem
- `6` = 16rem
- `7` = 32rem
- `8` = 64rem
- `9` = 128rem

chaining these together in this some order

`mt-0` = `margin-top: 0;`

`pt-0` = `padding-top: 0;`

### Examples of using heading classes with content

To maintain a hierachy of content, we often want to add a h1 styling to a lower content structure to maintain a good SEO performance, although this is a small ranking factor. It is also helpful for users of screenreaders so they're not jumping all over the page.

It is wise to keep this structure in place, a handy chrome extension to monitor this structure can be found here: <a href="https://chromewebstore.google.com/detail/afoibpobokebhgfnknfndkgemglggomo?utm_source=item-share-cb">HTML5 Outliner</a>

The below example is just for `h1` tags but can be applied to any content level block.
